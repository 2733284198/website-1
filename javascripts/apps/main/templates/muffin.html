Muffin

Muffin is a full-stack development tool for creating modern webapps. It addresses key issues such as the build process, module loading and package management at the proper level. Muffin also provides project templates, Rails-style full-stack generators, best practices for building Backbone apps, etc.

You can check out muffin.io for more details.

The project is open-sourced and hosted on GitHub.

P.S., this website is built with Muffin!
