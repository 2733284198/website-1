Muffin

Muffin is a full-stack development tool for creating modern webapps. It is designed with developers' productivity in mind.

Muffin elegantly addresses key issues such as the build process, module dependencies and package management. On top of that, Muffin also provides project templates for both client and server, Rails-style full-stack generators, best practices for building Backbone apps, etc. Check out muffin.io for more details.

Muffin is an open-source project hosted on GitHub. It's released under the MIT license.
